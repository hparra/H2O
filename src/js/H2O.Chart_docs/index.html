H2O.Chart(Boolean [scale_graph], Boolean [dotPlot], Boolean [show_line_number], String id, HEX [bgGradientStart], HEX [bgGradientEnd]) : static canvas

Create a new instance of H2O.Chart. Example: myChart = H2O.Chart( { id : "idName" , resize: true, ....}); To call function USE myChart.clear (); DO NOT USE myChart.self.clear(); All parameters passed to the contructor are optional. Except for id , id is required to be defined.

Boolean[scale_graph]Enable or disable scaling feature. Default set to enable.
Boolean[dotPlot]Enable or disable the rounded dot at each plot point. Default set to disable.
Boolean[show_line_number]Enable or disable the display of line numbers. Default is set to enable.
Stringidsets the id of the Chart canvas. **REQUIRED OPTION**
HEX[bgGradientStart]sets the beginning color of the background gradient. Default set to # 330033.
HEX[bgGradientEnd]sets the ending color of the background gradient. Default set to # 000033.
